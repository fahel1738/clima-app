# Meteora

Meteora é um aplicativo web de previsão do tempo com interface dinâmica e imersiva. A experiência muda visualmente conforme a cidade pesquisada, o clima atual e o período do dia, usando fundos animados inspirados em apps meteorológicos premium.

## Funcionalidades

- Busca de cidade com dados reais da API Open-Meteo
- Fundo dinâmico para sol, nuvens, chuva, neve, neblina e noite
- Temperatura atual, sensação térmica, umidade, vento, chuva e pressão
- Previsão para os próximos 7 dias
- Comparativo rápido entre hoje, amanhã e fim da semana
- Notícias e insights meteorológicos simulados para evolução futura
- Interface responsiva para desktop e mobile

## Tecnologias

- HTML5
- CSS3
- JavaScript ES6+
- Open-Meteo Geocoding API
- Open-Meteo Forecast API

## Objetivo

Criar uma aplicação intuitiva que vá além da previsão básica, combinando informação climática, visual dinâmico e microinterações elegantes.
